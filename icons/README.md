# QuickDefine Icons

This directory should contain the following icon files for the Chrome Extension:

## Required Icon Sizes

1. **icon16.png** - 16x16 pixels (toolbar icon)
2. **icon48.png** - 48x48 pixels (extension management page)
3. **icon128.png** - 128x128 pixels (Chrome Web Store)

## Design Guidelines

- **Style**: Minimalist, Apple-like design matching the extension's aesthetic
- **Color Scheme**: Use a clean, professional color palette (suggested: blue accent #007aff on white/light background)
- **Symbol**: Consider using a dictionary/book icon or a "D" letter with a modern, clean design
- **Format**: PNG with transparency support

## Icon Creation Tools

You can create these icons using:
- **Online Tools**: 
  - [Favicon.io](https://favicon.io/)
  - [IconGenerator](https://icon-generator.net/)
  - [Canva](https://www.canva.com/)
- **Design Software**: 
  - Adobe Illustrator
  - Figma
  - Sketch

## Quick Creation Method

1. Create a 128x128px icon design
2. Export as PNG
3. Resize to create the 48x48px and 16x16px versions
4. Ensure all icons maintain visual clarity at their respective sizes

## Placeholder Icons

If you need placeholder icons for testing, you can:
1. Create simple colored squares with text labels
2. Use any icon generator to create temporary icons
3. Replace them with final designs before publishing
